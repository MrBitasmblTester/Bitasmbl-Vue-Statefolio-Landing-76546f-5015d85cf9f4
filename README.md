# Bitasmbl-Vue-Statefolio-Landing-76546f-5015d85cf9f4

## Description
A modern single-page portfolio built with Vue.js that showcases projects, skills, and contact info while emphasizing client-side state management and reactive UI behavior for an intern-level front-end candidate.

## Tech Stack
- Vue.js

## Requirements
- Use a centralized Vue state store to manage theme, project filters, and portfolio data.
- Implement computed state (e.g., filtered projects) and ensure UI updates reactively.
- Control the contact form entirely with Vue state and show simple validation errors.
- Reflect the theme state on the root element to switch between light and dark styles.

## Implementation Steps
1. Initialize Vue single-page structure.
2. Create centralized Vue state store for theme, filters, and portfolio data.
3. Add computed properties for filtered projects.
4. Bind store state to UI for reactive updates.
5. Build contact form controlled entirely by Vue state with simple validation errors.
6. Bind theme state to root element classes to switch light/dark styles.
